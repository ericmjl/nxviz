# Version v0.7.6

This release introduces significant improvements to the build system, project configuration, and continuous integration workflows. It also includes enhancements to the codebase and documentation, ensuring a more streamlined development process and better project maintainability.

## New Features

- Added `__all__` attribute to `__init__.py` for explicit module exports, defining the public API and including major plot types and the version module. (5e1682) (Eric Ma)

## Bug Fixes

- Refined test parameters and assertions for polar-cartesian conversion to improve numerical stability and accuracy in angle comparison. (be22b3) (Eric Ma)
- Updated GitHub Actions workflow to use `GITHUB_TOKEN` for `gh-pages`, replacing `GHPAGES_TOKEN` and removing unnecessary configurations. (8ac7a5) (Eric Ma)

## Deprecations

- Removed the `Makefile` from the repository as tasks are now managed through alternative methods. (5cf4de) (Eric Ma)
- Deleted the `.editorconfig` file to simplify repository configuration. (60eda6) (Eric Ma)
- Removed unused configuration files `.pyup.yml` and `.readthedocs.yml` from the repository. (c2d6c3) (Eric Ma)

## Other Changes

- Switched the build system from `setuptools` to `hatchling`, updating build-system requirements and removing the license field from project metadata. (49da26) (Eric Ma)
- Updated the license field in `pyproject.toml` to use a string value 'MIT', aligning with standard practices. (19e155) (Eric Ma)
- Added `.env` file to `.gitignore` to prevent tracking of sensitive environment variables. (90e0de) (Eric Ma)
- Updated the Black formatter version in the pre-commit configuration. (b386c9) (Eric Ma)
- Updated release workflow to include `uv` binary installation and adjusted `pip install` command. (fad120) (Eric Ma)
- Updated CI workflow job name for clarity. (075516) (Eric Ma)
- Updated lockfile to reflect changes in dependency hashes. (9a2851) (Eric Ma)
- Refactored version management to a dedicated module, moving version information to a new `version.py` file. (1dbf05) (Eric Ma)
- Updated bumpversion configuration to use `pyproject.toml`, aligning with modern Python packaging standards. (fd671d) (Eric Ma)
- Updated PR checks workflow for improved concurrency and preview deployment. (46be12) (Eric Ma)
- Added workflow for releasing Python package to PyPI, automating the release process. (04aa76) (Eric Ma)
